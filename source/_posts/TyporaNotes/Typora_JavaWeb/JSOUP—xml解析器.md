# JSOUP

## 定义

Jsoup 是一个用 Java 语言编写的开源 HTML 解析库。它提供了一套非常方便的 API，用于从 URL、文件或字符串中提取和操作数据。Jsoup 实现了 WHATWG HTML5 规范，并能够将 HTML 解析为与现代浏览器相同的 DOM 结构。

## 分类和组成

1.Jsoup

工具类，可以解析html或xml文档，返回Document对象

2.Document

文档对象。代表内存中的DOM树

3.ELements

元素ELement对象的集合

4.Element

元素对象

5.Node

节点对象



## 使用方法

1.导入jar包

2.获取Document对象

3.获取对应标签的Element对象

4.获取数据
