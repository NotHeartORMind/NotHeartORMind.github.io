# MySQL入门

## 卸载：

1.卸载mysql软件

2.卸载C盘中的软件数据，否则下次无法安装成功。

## 服务启动：

一：手动打开 服务项目，开启MYSQL服务

二：通过cmd命令，net start mysql

## 服务关闭：

一：手动关闭服务项目，开启MYSQL服务

二：通过cmd命令，net stop mysql

## 登录：

mysql -uroot  -proot
