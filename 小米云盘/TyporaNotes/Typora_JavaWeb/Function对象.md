# Function对象

## 是什么

### 属性

length：

代表形参的个数

### 特点：

1.方法的定义，形参的类型不用写，返回值类型也不用写

2.方法是一个对象，如果定义相同名称的方法，会发生覆盖

3.在JS中，方法的调用只与方法的名称有关，和参数列表无关

4.在方法声明中，有一个隐藏的内置对象（数组），arguments，封装了所有的实际参数。
